# TAGGD SLA Dashboard - Tutorial Video Script (5 Minutes)

## Scene 1: Introduction & Dashboard Overview (0:00 - 0:45)
**Visual**: Dashboard homepage, showing all views
**Narration**: 
"Welcome to the TAGGD SLA Dashboard tutorial. This dashboard provides comprehensive tracking of Service Level Agreements and Key Performance Indicators across all your accounts. Let's explore the main features."

**Screen Actions**:
- Show dashboard loading with data
- Highlight the navigation tabs: FY Comparison, Project Analysis, Regional Performance
- Show the filter options at the top

## Scene 2: Account-Wise FY Comparison View (0:45 - 1:30)
**Visual**: FY Comparison table with RAG status
**Narration**:
"The FY Comparison view shows account performance year-over-year. Green indicates 75% or higher compliance, Amber is 50-74%, and Red is below 50%. You can see metrics like Time to Hire, Diversity targets, and Hit Ratios at a glance."

**Screen Actions**:
- Scroll through the FY Comparison table
- Highlight RAG status colors (Green, Amber, Red)
- Show the trend indicators (↑↓)
- Point to specific metrics

## Scene 3: Project Drill-Down Feature (1:30 - 3:00)
**Visual**: Click on a project (e.g., Siemens - GBS) to open drill-down
**Narration**:
"The most powerful feature is the project drill-down. Click any project name to see detailed monthly performance. Each performance measure shows the target, monthly scores from April to October, and year-to-date status. You can verify that all values match your Excel source exactly."

**Screen Actions**:
- Click on "Siemens - GBS" project
- Show drill-down modal opening
- Scroll through performance measures (Time to Hire, Source Mix, Diversity)
- Highlight:
  - Performance Measure column
  - Penalty/Non-Penalty status
  - Target values (e.g., "ER 15%", "70 Days")
  - Monthly scores with proper formatting (40%, 8%, 0%)
  - Met/Not Met status indicators (✓/✗)
  - YTD Score
- Close drill-down and open another project (e.g., SKF)

## Scene 4: Regional Performance & Charts (3:00 - 3:45)
**Visual**: Regional Performance view with charts
**Narration**:
"The Regional Performance view breaks down metrics by geography. Interactive charts show performance distribution and trends. You can filter by region, practice head, or specific metrics."

**Screen Actions**:

- Demonstrate filter functionality
